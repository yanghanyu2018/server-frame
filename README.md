后台服务端的框架
=================================================

说明
-------------------------------------------------
这个服务端的框架。
  
主要使用到``Node.Js``的如下组件:  

> * **express**：对外提供HTTP服务。  
> * **http**：对外提供HTTP服务。  
> * **net**：提供TCP服务链接。  
> * **mongodb**：数据的存储（v2.0以上）。  
> * **redis**：数据的存储（v2.0以上）。

**最后修改时间**：2016年11月25日

v1.0  ：初步的框架

开发模式

nodemon ./bin/www
